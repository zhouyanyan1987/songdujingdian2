### 3. 🔔 智能分章系统

#### 分章方式支持：

**1. 手动分章（优先级最高）**
```javascript
----- 第一章内容 -----
这是第一章的具体内容...

----- 第二章内容 -----
这是第二章的具体内容...

// 或者使用 === 符号
=== 第一节内容 ===
=== 第二节内容 ===
```

**2. 自动中文分章**
```javascript
// 按中文标点符号分章：
。！？； 这些符号作为分章依据
```

**3. 自动英文分章**
```javascript
// 按英文标点符号分章：
.!? 符号作为分章依据
```

**4. 智能句群分章**
```javascript
// 每3-5句自动组成一章
句1。句2。句3。 ← 组成第一章
句4。句5。句6。 ← 组成第二章
```

**5. 章节标记检测**
```javascript
// 自动识别章节标记：
第X章、第X篇、第X节、序言、结论等
```
