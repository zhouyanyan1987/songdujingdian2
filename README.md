# 经典诵读助手 - GitHub Pages 部署包

## 📖 项目简介

这是一个专为经典诵读而设计的Web应用，支持中华经典文献的学习和诵读跟踪。应用支持8部经典著作，具备智能分章、进度跟踪、100遍诵读成就系统等功能。

## ✨ 主要功能

### 📚 内置经典书籍
- **道德经** - 81章完整内容
- **易经** - 64卦精选内容  
- **黄帝内经** - 中医经典精选
- **弟子规** - 传统蒙学教育经典
- **诫子书** - 诸葛亮家书
- **大学** - 儒家四书之一
- **太上感应篇** - 道教经典
- **孝经** - 儒家孝道经典

### 🎯 核心功能
- **智能分章系统**：自动按句号分章，支持手动分章标记（-----、====）
- **进度跟踪**：详细的阅读进度显示和100遍诵读计数
- **成就系统**：完整的100遍诵读庆祝动画和成就展示
- **数据持久化**：所有数据保存在浏览器本地存储中
- **响应式设计**：完美适配手机、平板、电脑等各种设备
- **章节导航**：流畅的上一章/下一章切换
- **重读支持**：完成100遍后可继续重读并计数

### 🔧 技术特性
- **单文件应用**：所有功能集成在一个HTML文件中
- **零依赖**：无需外部库或框架
- **离线运行**：部署后完全离线使用
- **GitHub Pages优化**：专为GitHub Pages静态托管优化

## 📱 使用指南

### 开始阅读
1. 打开网页后，会自动加载8部经典书籍
2. 点击任意书籍卡片进入阅读模式
3. 在章节页面进行诵读和学习

### 进度管理
- **完成诵读**：点击"✅ 完成诵读"按钮
- **查看进度**：进度条和百分比实时显示
- **进度印章**：底部显示100个进度印章
- **重置进度**：如需重新开始可点击"🔄 重置进度"

### 添加自定义书籍
1. 点击"➕ 添加书籍"按钮
2. 输入书籍名称和内容
3. 系统会自动智能分章：
   - 使用 `-----` 或 `===` 作为手动分章标记
   - 自动按中文/英文标点符号分句
   - 每3-5句自动组成一章

### 智能分章示例
```
第一章内容在这里
-----
第二章内容在这里
-----
第三章内容在这里
```

## 🐛 故障排除

### 常见问题

**Q: 网页只显示标题，无法操作？**
A: 请确保：
- 文件名为 `index.html`（大小写敏感）
- 浏览器支持JavaScript
- 网络连接正常（首次加载需要）

**Q: 数据丢失了怎么办？**
A: 应用数据保存在浏览器本地存储中：
- 清除浏览器数据会导致数据丢失
- 建议定期导出重要数据
- 同一浏览器/设备的数据不会丢失

**Q: 如何备份数据？**
A: 打开浏览器开发者工具：
```javascript
// 在控制台执行：
console.log(JSON.parse(localStorage.getItem('books')))
```

**Q: 分章不准确怎么办？**
A: 使用手动分章标记：
```
内容1
-----
内容2
-----
内容3
```

**Q: 在手机上显示不正常？**
A: 确保：
- 使用现代浏览器（Chrome、Safari、Firefox等）
- 网络连接正常
- 清除浏览器缓存后重试
