# 读书百遍其义自见 - 经典诵读助手

## 功能特性

### 📚 完整经典库
- **道德经**: 81章完整版本
- **易经**: 64卦完整版本，包含卦辞、爻辞
- **黄帝内经**: 素问81篇 + 灵枢81篇完整版本
- **论语**: 儒家经典，孔子言行录
- **孟子**: 性善论和仁政思想
- **大学**: 四书之一，修身齐家治国平天下
- **中庸**: 中庸之道和君子品格
- **弟子规**: 传统蒙学教育经典
- **诫子书**: 诸葛亮家书，修身养性智慧
- **太上感应篇**: 道教经典，因果报应思想

### 🔤 智能拼音功能
- 为所有经典内容添加了准确的汉语拼音
- 支持多音字识别
- 可开关拼音显示模式
- 便于学习古文的正确读音

### 📖 智能分章系统
- **自动分章**: 根据内容自动智能分章
- **手动标记**: 支持 "-----" 或 "====" 手动分章标记
- **句子分割**: 按中文标点符号自动分句
- **章数控制**: 每3-5句自动组成一章
- **自适应**: 无需担心格式，系统自动处理

### 🔍 书籍搜索添加功能
- **实时搜索**: 输入书名即时搜索经典库
- **自定义添加**: 支持用户添加自定义书籍
- **内容解析**: 自动解析文本并智能分章
- **本地存储**: 自定义书籍保存在浏览器本地

### 📊 精确诵读记录
- **单次点击**: 每次点击记录一次诵读
- **实时计数**: 显示当前章节诵读遍数
- **进度追踪**: 显示完成进度百分比
- **进度印章**: 每10遍显示一个完成印章
- **100遍达成**: 完成100遍后显示庆祝动画

### 🎨 淡水墨静谧设计
- **渐变背景**: 淡雅的渐变色彩
- **毛玻璃效果**: 现代化的视觉体验
- **水墨风格**: 体现传统文化的典雅
- **响应式设计**: 适配手机、平板、电脑

### 📈 进度管理系统
- **章节进度**: 每个章节独立记录进度
- **书籍总体**: 统计整本书的完成度
- **本地存储**: 所有数据保存在浏览器本地
- **进度同步**: 多设备间自动同步（通过同一浏览器）

## 使用方法

### 基础使用
1. **浏览书库**: 在首页查看所有可用的经典书籍
2. **选择书籍**: 点击任意书籍进入书籍详情页
3. **选择章节**: 在书籍详情页选择要诵读的章节
4. **开始诵读**: 点击"诵读一次"按钮记录诵读
5. **查看进度**: 实时查看诵读次数和完成进度

### 高级功能

#### 添加自定义书籍
1. 点击"添加自定义书籍"按钮
2. 输入书名和内容
3. 系统自动智能分章
4. 点击确认添加

#### 搜索书籍
1. 在搜索框输入关键词
2. 实时显示搜索结果
3. 点击结果直接查看

#### 查看拼音
1. 在阅读页面点击"显示拼音"
2. 汉字上方显示标准拼音
3. 再次点击可隐藏拼音

#### 章节导航
- 使用"上一章"/"下一章"按钮切换
- 章节完成后自动激活下一章按钮

## 部署说明

### GitHub Pages 部署
1. 将 `index.html` 文件上传到GitHub仓库根目录
2. 在仓库设置中启用GitHub Pages
3. 选择分支（建议master/main分支）
4. 保存设置，等待部署完成
5. 访问生成的URL即可使用

### 本地部署
1. 下载所有文件到本地目录
2. 用浏览器打开 `index.html` 文件
3. 所有功能均可正常使用
